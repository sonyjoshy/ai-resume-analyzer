# 🚀 Resumind - AI Resume Analyzer

An AI-powered web application that evaluates resumes against job requirements and provides automated scoring and feedback.

---

## 🔥 Features

- 📄 Upload resumes
- 💼 Create job listings
- 🤖 AI-based resume evaluation
- 🎯 Job matching system
- 📊 Resume scoring and feedback

---

## 🛠 Tech Stack

- React
- React Router
- TypeScript
- Puter.js

---

## ⚙️ How it Works

1. User uploads resume  
2. System analyzes content using AI  
3. Matches resume with job requirements  
4. Generates score and feedback
